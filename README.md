## shortenURL

PHP驱动的短网址API

## Description

因为众所周知的我太菜的原因，这个的代码实现十分粗暴，还写的特别丑。

## Code

`-1`：没有传入网址

`114`：生成成功

`514`：不支持生成此域名的短链接

`404`：进行缩短的网页不存在
